# 日历应用功能总览

## 🎯 项目概述

这是一个基于 Tauri + Vue 3 + Ant Design 开发的跨平台桌面日历应用。

## 📝 配置说明

### 天气API配置

1. 注册高德开放平台账号
2. 获取API Key
3. 在 `src/components/RightPanel.vue` 中替换 `YOUR_AMAP_KEY`
4. 详细步骤见 `WEATHER_API_GUIDE.md`

### 城市配置

编辑 `RightPanel.vue` 中的城市代码：

```javascript
const adcode = '110000'; // 北京
```

常用城市代码：

- 北京：110000
- 上海：310000
- 广州：440100
- 深圳：440300
